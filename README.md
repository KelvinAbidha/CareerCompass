# Career Compass MVP

This is a simple MVP of a Career Compass application that allows users to log their daily activities.

## Running the application

1.  Open a terminal and navigate to the project directory.
2.  Run `npm install` to install the dependencies.
3.  Run `npm start` to start the server.
4.  Open the `index.html` file in your browser.

## How to use

1.  The application will automatically sign you in as an anonymous user.
2.  Fill in the form to log your daily activity.
3.  The logs will be displayed in the "This Week's Logs" section.
4.  The "Generate Post" button will be enabled when there is at least one activity logged for the current week.
